# 🛡️ Lego-Auth-System
A sleek and responsive Login & Registration system built with a modular approach. This project demonstrates front-end fundamentals with a focus on smooth UI transitions and secure-ready input structures.

## 🚀 Features
- **Dynamic Form Switching:** Seamless toggle between Login and Register using JavaScript.
- **Glassmorphism Design:** Modern UI with CSS blur and transparency effects.
- **Form Validation:** Client-side validation for emails and passwords.

## 🛠️ Tech Stack
- **HTML5:** Structure.
- **CSS3:** Styling & Animations.
- **JavaScript:** DOM Manipulation.

## 🔒 Cybersecurity Context
This project serves as a foundation for understanding the **Client-Side Document Object Model (DOM)**, which is essential for identifying vulnerabilities like **XSS** and implementing robust **Input Sanitization**.
